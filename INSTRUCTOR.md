## ✅ เตรียมก่อนสอน
1. ให้ผู้เรียนติดตั้ง **Docker Desktop** + **Claude Code** มาก่อน (โมดูล 0)
2. ทดสอบรันเองครั้งหนึ่ง:
   ```bash
   make up && make load && make verify
   ```
   ควรขึ้น "🎉 พร้อมแล้ว!" ทุกบริการ
3. เปิด `slides.html` เลือกภาษา (EN/TH/中文) ให้ตรงกลุ่มผู้เรียน

## 🧯 ปัญหาที่เจอบ่อย
| อาการ | วิธีแก้ |
|-------|--------|
| ElasticSearch ขึ้น unhealthy / OOM | เพิ่ม RAM ให้ Docker หรือคงค่า `ES_JAVA_OPTS=-Xms512m -Xmx512m` |
| พอร์ตชนกัน (5432/8080/...) | แก้ฝั่งซ้ายของ `ports:` ใน `docker-compose.yml` |
| ข้อมูลไม่โหลดใน Postgres/Mongo | ข้อมูล seed รันแค่ตอนสร้าง volume ครั้งแรก → `make reset` |
| Airbyte ต่อ Postgres ไม่ติด | ใช้ host `host.docker.internal` ไม่ใช่ `localhost` |

## 💡 เคล็ดลับการสอนด้วย Claude Code
- ให้ผู้เรียน **ลองเขียนเองก่อน** แล้วค่อยให้ Claude ช่วย — จะได้ฝึกคิด
- เน้นให้ถาม Claude แบบ "อธิบายทีละบรรทัด" เพื่อเข้าใจ ไม่ใช่แค่ copy
- โชว์การ debug จริง: จงใจทำ error แล้วให้ Claude ช่วยอ่าน log

## 🧹 หลังจบคลาส
```bash
make down      # ปิด (เก็บข้อมูล)
make reset     # หรือเริ่มใหม่สะอาดๆ สำหรับรอบถัดไป
```
