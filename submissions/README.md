# 📥 ส่งงาน (Submissions)

ผู้เรียนแต่ละคนสร้างโฟลเดอร์ของตัวเองที่นี่ แล้วเก็บงานโปรเจกต์จบ (โมดูล 8)

```
submissions/
  └── your-name/
        ├── README.md      # อธิบายว่าทำอะไร เรียนรู้อะไร
        ├── postgres.sql   # query ที่เขียนเอง
        ├── mongodb.js
        └── elastic.http
```

## เกณฑ์คร่าวๆ
- เลือกทำอย่างน้อย 3 ข้อจากโจทย์ในโมดูล 8
- โค้ดรันได้จริงกับ stack ของ workshop
- เขียนสรุปสั้นๆ ว่าใช้ Claude Code ช่วยตรงไหนบ้าง

> เริ่มไม่ถูก? เปิด Claude Code แล้วถาม: *"ช่วยแนะนำหัวข้อโปรเจกต์จบที่เหมาะกับผมหน่อย"*
