# โมดูล 8 — โปรเจกต์จบ: สร้าง Mini Data Platform ของคุณเอง

## 🎯 เป้าหมาย
รวมทุกอย่างที่เรียนมา สร้างระบบข้อมูลร้านค้าเล็กๆ ของตัวเอง โดยมี Claude Code เป็นคู่หู

## 📋 โจทย์
ขยายร้านค้าออนไลน์ของเราให้ฉลาดขึ้น เลือกทำอย่างน้อย **3 ข้อ** จากนี้:

### ระดับพื้นฐาน
- [ ] เพิ่มหมวดสินค้าและสินค้าใหม่อย่างน้อย 5 รายการใน PostgreSQL
- [ ] เขียน SQL รายงานยอดขายรวมแยกตามหมวดสินค้า
- [ ] เพิ่มรีวิวใหม่ใน MongoDB แล้วหาคะแนนเฉลี่ยของสินค้าที่เพิ่ม

### ระดับกลาง
- [ ] สร้างหน้าค้นหาด้วย ElasticSearch ที่ค้นได้ทั้งชื่อและคำอธิบาย พร้อมกรองตามราคา
- [ ] ใช้ Airbyte sync ข้อมูล `customers` หรือ `orders` ออกไป destination
- [ ] เขียน aggregation ใน MongoDB หา top reviewer (คนรีวิวเยอะสุด)

### ระดับท้าทาย
- [ ] สร้าง view รวมข้อมูลสินค้า (Postgres) + คะแนนรีวิว (Mongo) + ผลค้นหา (Elastic)
- [ ] เขียนสคริปต์ตรวจ data quality (เช่น สินค้าที่ stock ติดลบ, ออเดอร์ไม่มี item)
- [ ] เพิ่ม dataset ของคุณเอง (เปลี่ยนธีมร้านเป็นอย่างอื่นก็ได้!)

## 🤖 ใช้ Claude Code อย่างไรให้คุ้ม
- ให้มัน **ออกแบบ** ก่อนลงมือ: *"ผมอยากทำรายงานยอดขายรายเดือน ควรเขียน query ยังไง"*
- ให้มัน **เขียนโค้ด** แล้วเราตรวจ: *"เขียน SQL ตามนี้ให้หน่อย..."*
- ให้มัน **debug**: *"query นี้ขึ้น error ช่วยดู"*
- ให้มัน **อธิบาย**: *"อธิบายว่าโค้ดที่เขียนให้ทำงานยังไงทีละบรรทัด"*

## 🏁 ส่งงาน
- บันทึก query / สคริปต์ที่เขียนไว้ในโฟลเดอร์ `submissions/<ชื่อคุณ>/`
- เขียน README สั้นๆ ว่าทำอะไรไปบ้าง และเรียนรู้อะไร

และที่สำคัญ — ได้ฝึกทำงาน data จริงโดยมี **Claude Code** เป็นผู้ช่วย 🎉
